## Reference

### Badge

#### Component

Custom component to replace the badge outer component.

| Type            | Default                                           |
| --------------- | ------------------------------------------------- |
| React Component | `Press handlers present then Pressable else View` |

---

#### badgeStyle

Additional styling for badge (background) view component.

| Type               | Default |
| ------------------ | ------- |
| View style(Object) | None    |

---

#### containerStyle

Style for the container.

| Type               | Default |
| ------------------ | ------- |
| View style(Object) | None    |

---

#### onLongPress

Called when a long-tap gesture is detected.

| Type                         | Default |
| ---------------------------- | ------- |
| GestureResponderEventHandler | None    |

---

#### onPress

Called when a single tap gesture is detected.

| Type     | Default |
| -------- | ------- |
| Function | None    |

---

#### onPressIn

Called when a touch is engaged before `onPress`.

| Type                         | Default |
| ---------------------------- | ------- |
| GestureResponderEventHandler | None    |

---

#### onPressOut

Called when a touch is released before `onPress`.

| Type                         | Default |
| ---------------------------- | ------- |
| GestureResponderEventHandler | None    |

---

#### pressableProps

| Type                                 | Default |
| ------------------------------------ | ------- |
| PressableProps except click handlers | None    |

---

#### status

Determines color of the indicator.

| Type                                           | Default |
| ---------------------------------------------- | ------- |
| "primary" or "success" or "warning" or "error" | primary |

---

#### textProps

Extra props for text component.

| Type      | Default |
| --------- | ------- |
| TextProps | None    |

---

#### textStyle

Extra styling for icon component.

| Type               | Default |
| ------------------ | ------- |
| Text Style(Object) | None    |

---

#### value

Text value to be displayed by badge, defaults to empty.

| Type            | Default |
| --------------- | ------- |
| React.ReactNode | None    |

---
